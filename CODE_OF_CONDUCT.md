# 行動規範 (Code of Conduct)

## 私たちの誓い

メンバー、コントリビューター、およびリーダーである私たちは、年齢・体格・障がいの有無・
民族性・性自認や性表現・経験の水準・教育・社会経済的地位・国籍・容姿・人種・宗教・
性的指向にかかわらず、誰もがハラスメントのないコミュニティへ参加できることを誓います。

## 行動の基準

コミュニティにとって好ましい行動の例:

- 他者への共感と思いやりを示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを行い、また快く受け入れる
- 自らの過ちに責任を持ち、影響を受けた人々へ謝罪し、その経験から学ぶ
- 個人だけでなくコミュニティ全体にとって最善となることに目を向ける

許容されない行動の例:

- 性的な表現・誘いかけ、その他あらゆる性的な注目や接近
- 荒らし、侮辱・中傷的なコメント、個人攻撃や政治的攻撃
- 公的・私的を問わないハラスメント
- 明示的な許可なく他者の私的情報（物理的・電子的な住所など）を公開すること
- 職業上不適切とみなされるその他の行為

## 適用範囲

本行動規範は、コミュニティのあらゆる空間に適用され、また個人がコミュニティを公的に
代表する場合にも適用されます。

## 執行

不適切な行為を目撃した、または受けた場合は、プロジェクトメンテナ
（hironobunigo@gmail.com）まで報告してください。すべての報告は速やかかつ公正に
レビューされ、調査されます。メンテナは報告者のプライバシーと安全を尊重する義務を負います。

## 帰属

本行動規範は [Contributor Covenant](https://www.contributor-covenant.org) バージョン 2.1 を
基にしています。
