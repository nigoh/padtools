Juml ドキュメント

Java / Android（Gradle + AndroidManifest.xml + AIDL）プロジェクトを解析し、PlantUML 経由で UML 図を生成する Swing + CLI ツール。

Version 2.1 Java 17+ License MIT 図種 15 Graphviz 不要（Smetana 同梱）

クイックスタート

# GUI を起動（引数にプロジェクトを渡すと起動時に解析）
java -jar Juml.jar ~/AndroidStudioProjects/MyApp

# CLI でクラス図を SVG 出力
java -jar Juml.jar -c -o class.svg ~/AndroidStudioProjects/MyApp

# 全成果物を一括出力
java -jar Juml.jar --all -o ./out ~/AndroidStudioProjects/MyApp

このサイトについて

Juml のドキュメントは 「リッチな HTML を成果物にする」アプローチに倣い、すべて HTML に統一しています。各ページは共有スタイル（assets/style.css / assets/app.js）を読み込み、サイドバーナビ・スクロールスパイ・コピーボタン・CLI オプションのライブ絞り込みを備えます。GitHub Pages でそのまま公開できます。

Claude Code 連携: AOSP / AAOS 解析用のエージェント・スキル・スラッシュコマンドはリポジトリの .claude/ 配下に置いています。カタログと使い方は Claude Code 連携ページに、導入手順は .claude/SETUP_CLAUDE_AGENTS_SKILLS.md にまとめています。
