これは何？

Juml 2.1 は、Java / Android（Gradle + AndroidManifest.xml + AIDL）プロジェクトを解析し、PlantUML 経由で UML 図を生成する Swing ベースの対話型ツールです。GUI と CLI の両方を備え、PlantUML（Smetana レイアウト）を同梱しているため Graphviz の追加インストールは不要です。

  • バージョン: 2.1（旧 1.x の PAD/SPD 機能は廃止し、UML 専用ツールへ完全転換）
  • ライセンス: MIT（同梱 PlantUML は GPLv3 ではなく MIT 版アーティファクト plantuml-mit）
  • 必要環境: Java 17 以上（Windows / macOS / Linux）

主な機能

区分内容
UML クラス図パッケージ単位グループ化 / 継承・実装・利用関係 / AAOS パターン / Manifest 連携（<<Activity>> 等）/ JavaDoc・アノテーション・enum 定数
シーケンス / アクティビティ図Class.method 起点で多段トレース。if/while/switch/try を alt/loop/group に変換
パッケージ / 共通 / 継承 / モジュール図パッケージ集約・fan-in 上位・継承ツリー・JPMS（module-info）
コンポーネント / Manifest / Deep Link 図Activity/Service/Receiver/Provider、exported、uses-permission、VIEW+BROWSABLE
Gradle 依存 / Soong 図モジュール間 + Maven 依存、libs.versions.toml 解決 / Android.bp 依存
解析機能影響解析（impact）/ 逆参照（ref-find）/ 関数差分（func-diff）/ 画面遷移 / Room ER / VHAL フロー / AIDL バインディング
GUI プレビューApache Batik による ベクター SVG プレビュー（PNG の 4096×4096 制約を受けない）
エクスポートSVG / PNG / PUML / Markdown サマリーを出力、--all で一括出力

全機能の詳細は 開発者リファレンスを参照してください。

プロジェクト構成（ソース）

src/main/java/juml/
├── Main.java                   # CLI / GUI のエントリポイント
├── app/uml/                    # UML 専用 Swing GUI（UmlMainFrame, DiagramTabPane ほか）
├── app/cli/                    # CLI ディスパッチと各種コマンド
├── core/formats/
│   ├── java/                   # Java/AIDL 字句解析、JavaParser フロントエンド
│   ├── uml/                    # PlantUML 生成器（クラス/シーケンス/…）、AST 抽出、SQLite db
│   └── android/                # Gradle / Manifest / Version Catalog / layout / nav パーサ
├── core/refs, impact, dataflow, funcdiff, screen, aaos, aosp
└── util/                       # OptionParser / Messages / PathUtil 等
  • 本体: 256 ファイル / 約 48.6k LoC
  • テスト: JUnit で 136 ファイル / 約 26k LoC（約 1,553 テストケース）

ビルドと実行

./gradlew test      # ユニットテスト
./gradlew check     # テスト + Checkstyle (maxWarnings = 0)
./gradlew jar       # build/libs/Juml.jar（依存内包 fat jar）
./gradlew makeZip   # 起動補助スクリプト付き配布 zip
# GUI 起動（引数なしで UML GUI が立ち上がる）
java -jar Juml.jar [プロジェクトディレクトリ]

# CLI で全成果物を一括出力
java -jar Juml.jar --all -o ./out ~/AndroidStudioProjects/MyApp
